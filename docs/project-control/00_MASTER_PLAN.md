# Master Jam Master Plan

آخرین به‌روزرسانی: 2026-06-28

## هدف کلان سیستم‌عامل هوشمند مستر جم

مستر جم باید به یک سیستم‌عامل مدیریتی یکپارچه برای کنترل عملیات، نیروی انسانی، مالی، تولید، انبار، تجربه کاربری، موبایل و داده تبدیل شود. هر شاخه می‌تواند مستقل رشد کند، اما همه شاخه‌ها باید در نهایت به کابین مرکزی کنترل وصل شوند تا مدیر تصویر واحد، قابل اعتماد و قابل اقدام از وضعیت کسب‌وکار داشته باشد.

هدف کلان فقط ساخت چند صفحه جدا نیست؛ هدف ساخت یک «کابین فرمان» است که تصمیم‌ها، هشدارها، گزارش‌ها، baselineها، تغییرات و ریسک‌ها را در یک معماری قابل توسعه نگه دارد.

## ماژول‌های مادر

| کد | نام | نقش |
|---|---|---|
| CORE | هسته مرکزی | shell، route، کنترل پروژه، کابین مرکزی، قراردادهای مشترک |
| WF | نیروی انسانی | برنامه هفتگی، فضاها، کارمندان، تحلیل زمان/مکان، drift و readiness |
| FIN | مالی | هزینه، فروش، سود، جریان نقد، گزارش مالی |
| PROD | تولید | برنامه تولید، ظرفیت تولید، کنترل کیفیت، سفارش‌ها |
| INV | انبار | موجودی، ورودی/خروجی، حد سفارش، مغایرت‌ها |
| UI | طراحی و تجربه کاربری | زبان بصری، RTL، dark mode، UX مدیریتی |
| MOBILE | اپ موبایل | تجربه همراه مدیر/کارمند، اعلان‌های سبک، مشاهده سریع |
| DATA | استخراج داده و محک و بانک | ETL، ورود/خروج داده، تطبیق بانک/محک، کیفیت داده |
| CONTROL | مرکز کنترل | تصمیم‌گیری، اولویت‌بندی، کنترل تغییرات، handoff و audit |

## قانون اتصال به کابین مرکزی

1. هیچ شاخه‌ای نباید جزیره مستقل دائمی بماند.
2. هر route، service، analyzer یا storage key جدید باید مشخص کند به کدام شاخه مادر وصل است.
3. خروجی هر شاخه باید در آینده قابل نمایش در کابین مرکزی باشد: KPI، alert، report، status یا decision item.
4. تغییرات حساس باید در اسناد کنترل پروژه ثبت شوند.
5. route، storage key، مدل داده و migration بدون تأیید مرکز کنترل تغییر نمی‌کند.
6. هر P باید پایان مشخص داشته باشد: فایل‌ها، اثرات، تست/build، ریسک‌ها و پیشنهاد قدم بعد.

## زیرپروژه‌های ثبت‌شده اما merge نشده

- `FIN-AUDIT`: `audit-app` / داشبورد بحران نقدینگی / Subproject
- `DATA-MAHAK`: `mahak-web-version` / ثبت کالا و خروجی محک / Subproject

این زیرپروژه‌ها فعلاً merge نمی‌شوند. فقط ایده‌ها، schemaها، مدل‌ها و منطق‌های مفید آن‌ها بعداً با تأیید مرکز کنترل قابل استخراج هستند.
